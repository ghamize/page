What is HTML?

HTML is a "markup language". We place <> between things in HTML, to give it meaning and to create structure. Your web page is built on HTML - it is the language of a web page.
Tags are used to create HTML elements. For example, two "< p >" tags create a paragraph element, where you can write in your text. Elements are building blocks in HTML, and help you identify groups of information and to style these groups of information.

    WHY DO WE NEED TAGS?

    You don't actually need tags to display anything in the browser, but these tags show the browser what this content on your page means. It also helps you organize it. For example, adding < p > < / p > (no spaces) allows your browser to know that the content in-between is a paragraph. You can then change the paragraph's size, positioning, colour, and all of that because you have tags around it. Without the tags, you can't do anything to it. This is when CSS becomes important.

WHAT IS CSS?

CSS is a short term for "cascading style sheets". This is what makes pages look pretty. Anything from changing the font colour to the background colour or adding cool effects is done in a CSS page. CSS is coded in using the same program, but it's a different language.
